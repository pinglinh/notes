## Markdown syntax highlighting

You can render syntax highlighting in a codeblock by stating which language it is next to first three backticks such as:

````
```js
const sayHello = () => {
  console.log("Hello!")  
}

sayHello()
```
````

````
```ruby
def say_hello
  puts "Hello!"
end
```
````

For language list which is supported visit this [website](https://support.codebasehq.com/articles/tips-tricks/syntax-highlighting-in-markdown).
